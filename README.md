## ✨ Features that redefine recruitment

### For Job Seekers 👨‍💻
*   **🧠 AI Match Scoring:** Instantly see your match percentage (75%-95%) based on deep semantic analysis of your skills against job requirements.
*   **🤖 Live AI Career Coach & Negotiator:** Chat with an integrated AI headhunter for salary negotiation, interview prep, and career advice.
*   **📄 AI Resume Studio:** Auto-parse your resume (PDF/DOCX), generate cover letters, and get ATS optimization scores.
*   **🌙 Dynamic Theme:** Ultra-premium UI with automatic system dark/light mode detection and glassmorphic micro-animations.

### For Employers 🏢
*   **📊 Employer Copilot:** An embedded AI assistant to help you write job descriptions, summarize applicant profiles, and generate hiring analytics.
*   **🔍 Semantic Search:** Search candidates by *intent*, not just keywords. (e.g., "Find me a backend dev with payment gateway experience").
*   **⚡ Real-Time Dashboards:** Track applications and schedule live interviews with seamless Socket.io real-time updates.

| Layer | Technologies |

## 🚀 Quick Setup Guide

### 1. Clone the repository
```bash
git clone https://github.com/Allen73737/job_portal07.git
cd job_portal07
```

### 2. Backend Setup
```bash
cd Backend
npm install
```
Create a `.env` file in the Backend directory with your credentials:
```env
MONGO_URI="your_mongodb_connection_string"
GROQ_API_KEY="your_groq_api_key"
PORT=5005
```
Start the backend server:
```bash
npm run dev
```

### 3. Frontend Setup
```bash
cd ../Frontend
npm install
npm run dev
```
